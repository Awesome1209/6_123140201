# Awi201News - Tugas 6

Aplikasi berita berbasis Compose Multiplatform yang mengambil data dari public API dan menampilkan:
- list artikel dengan image, title, dan description
- detail artikel saat item di-tap
- pull to refresh
- loading, success, dan error states
- halaman profile reuse dari tugas sebelumnya

## API
- NewsAPI `top-headlines`
- API key telah disesuaikan di project

## Fitur
- Bottom bar: News, Profile
- Filter kategori
- Search kategori / title / description
- Pull to refresh
- Repository pattern
- Detail screen
